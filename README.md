## Servicios Disponibles
### Recepción Solicitud
* **Funcion:** Solicitud de servicio por parte del cliente

## Ejecutar Servicios
* Sin _daemon_
```
cd Solicitud
node recibirSolicitud.js
```
* Con _daemon_
```
nodemon recibirSolicitud.js
```

## Otra información
* Instalar Express
```
npm install --save express
```
* Instalar Axios
```
npm install --save axios
```
